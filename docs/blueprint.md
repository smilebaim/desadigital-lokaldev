# **App Name**: Nusantara Canvas

## Core Features:

- Full-Screen Interactive Map: An immersive, high-resolution SVG and Canvas-based map of Indonesia that allows users to navigate across provinces with smooth pan and zoom.
- AI Province Storyteller: A generative AI tool that analyzes selected regions to provide rich, narrative insights into local history, culture, and hidden gems.
- Island-Hopping Discovery Panel: A sliding dynamic overlay providing localized key metrics and highlights for the currently focused island or province.
- Archipelago Atmospheric Engine: An AI-powered system that shifts the map's visual ambiance in real-time based on the time of day and regional weather data.
- Global Search Navigation: A fast search utility to fly the camera perspective directly to any major Indonesian city or landmark.

## Style Guidelines:

- The palette is inspired by 'Maritime Heritage,' utilizing a dark scheme to allow map visuals to pop. Primary: Oceanic Cobalt (#2F61FF). Background: Deep Midnight Navy (#0A0C14). Accent: Horizon Cyan (#00D1FF).
- A font pairing is used to balance tech and legibility. 'Space Grotesk' for tech-oriented headlines and navigation labels (sans-serif); 'Inter' for UI descriptions and detailed text blocks (sans-serif).
- Ultra-minimalist monoline icons representing geographic landmarks and cultural symbols, ensuring visual clarity on complex map backgrounds.
- Full-viewport edge-to-edge map canvas with glassmorphism-based UI overlays pinned to corners for an unobscured view.
- Smooth inertia-based camera movements and subtle topographic ripple effects when regions are selected.
